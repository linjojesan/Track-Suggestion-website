Design and CSS Track

CSS is the main language of design, it includes colors, layout, fonts, etc. The three main languages involved are HTML which is used for the infrastructure of the page, CSS, and JavaScript which is nuts and bolts or functionality of the website. If you are interested in creating amazing looking webpages creating efficient and flawless navigation infrastructures than this is the track for you.

Ruby

Ruby on Rails is the language that such famous applications like airbnb, Shopify, SoundCloud, Hulu, and even Github were built upon. It is the language of speed, and has been in development since 2004. Ruby makes coding easier but still requires a strong knowledge of other languages to use efficiently.

PHP

PHP, is the language of Wordpress, Drupal, and Jumal, it also is the language that programs 80% of the worlds serverside applications. Is is very useful for content heavy sites, and very useful for web development in general. The government runs on PHP.

C#

The C# language was developed by Microsoft and is a "simple, modern, general purpose, object oriented language. It is the goto for large companies, and one of the most in-demand languages in the job market. It is mostly used with business applications.

Android

If working on the latest mobile apps is your thing than learning the android tract is the way to go. You will be learning the latest in mobile development, languages like swift that power the iphone and ipad, and the language of Android which will allow you to have the needed skills to create beautiful and efficient mobile applications.
